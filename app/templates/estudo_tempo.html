O que é Takt Time?
Takt Time (s/peça) é o tempo teórico para bater a meta por hora: takt = 3600 / UPH Meta.
Como você usa Perda padrão, o sistema também calcula um Tempo alvo com perda (mais apertado) que é o que define OK/BALANCE.
Quando uma operação fica com UPH Real < UPH Meta, marcamos como BALANCE e a linha fica destacada.
Dica: UPH Real = (3600 / TempoCiclo) * (1 - perda). BALANCE aparece quando UPH Real < UPH Meta.
